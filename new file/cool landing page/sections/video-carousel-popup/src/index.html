Image Carousel with video popup/modal

I am currently working on a shopify website and did not want to use a shopify plugin so I set out to code my own! Leaving a little bit of instructions here for anyone else who would like to use this as well. Happy coding beautiful people!

Sources

To properly install the 3rd party scripts, I used the most lightweight options I could find online. Removing jquery completely and using javascript would help reduce page load time for anyone who needs it.

Splide.js (image carousel)

https://splidejs.com/

This is a fantastic carousel script that allows much more optimization than just what is here. You can checkout their website for more information and optimization.

new Splide( '#image-carousel' ).mount();

Lity (video modal)

https://sorgalla.com/lity/

Lity is a ultra-lightweight, accessible and responsive lightbox plugin which supports images, iframes and inline content out of the box. Minified and gzipped, its total footprint weights about 3kB. Requires jQuery.
